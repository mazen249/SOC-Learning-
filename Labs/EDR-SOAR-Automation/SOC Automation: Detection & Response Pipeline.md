# 🛡 SOAR + EDR Home Lab
### Building a Detection & Automation Workflow using LimaCharlie, Tines, Slack, and Docker

---

# 📌 Overview

In this project, I built a simple SOC/SOAR + EDR Home Lab to understand how modern security teams monitor endpoints, detect suspicious activity, and automate alerts.

The lab focuses on:
- Endpoint Detection & Response (EDR)
- Threat Detection
- Security Automation
- Alerting Workflows
- Incident Investigation
